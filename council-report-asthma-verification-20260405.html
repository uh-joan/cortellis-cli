Blind Spots the Council Caught

Asthma Is the Easy Case

Mature indication, consolidated leaders, stable MoA landscape. The skill has not been stressed against fragmented indications (IPF, rare CNS, early I-O combos) where the tier system will either hold or collapse into noise.

The GSK-Exit Insight May Be a Model Artifact

Domain Expert C: Real pharma divestments go to specialty buyers (Covis, Cipla), not to the company with the most overlapping mechanisms. The headline insight is mechanism-overlap math, not divestment logic.

No Reproducibility Check

If the same asthma query tomorrow returns different CPI tiers due to upstream data shifts, the "deterministic footer" is a lie. Nobody tested this.

No User Model

BD analyst doing weekly scans? VP prepping a board deck? Consultant billing hours? The right scenario library, drill-down depth, and validation UX depend on who uses this.

65% Baseline Is Self-Referential

Set by a previous council pass using similar rubrics. The 65→78 delta is movement within the same measurement frame, not independent ground truth. Some of the "improvement" is the skill learning the test.
