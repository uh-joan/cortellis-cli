## v0.10 implications

1. **[P0] Raw-data-pull step missing from the trial workflow.** The nomination-to-trial sequence has no mechanism for Joan to pull raw data for a newly nominated indication before the session. This must be resolved before any production trial. Proposed fix: add an explicit step between T-48h nomination and T-0 trial — "Joan confirms indication is covered by existing raw data OR executes `/landscape <indication>` to pull it and stages output 24h before session." This is a workflow gap, not a scoring gap, but it is the most operationally critical defect surfaced by this dry-run.

2. **[P1] Program-level pipeline table missing from output.** Carlos's complaint #2: the tool surfaces competitive programs by name but does not present them in a structured table (asset, sponsor, phase, primary indication, expected readout, overlap with nominated indication). `strategic_briefing.md` and `scenario_library.md` are narrative-first. A BD decider doing a term-sheet evaluation needs a scannable program table, not a prose competitive summary. This is a v0.10 output-format scope item.

3. **[P1] Deal-comp triangulation methodology missing.** When the tool finds sparse deal comps for a rare indication, it should not stop at "comps are thin." It should offer a triangulation path: rare-indication comps from adjacent indications, mechanism-class comps (e.g., TEAD/Hippo deals across all indications), and asset-stage comps (Phase 2-ready small-molecule in rare oncology). Carlos's complaint #3 is a feature gap, not a data gap.

4. **[P2] No "start here" navigation guidance across the three output files.** Carlos spent ~90 seconds deciding where to begin. The three-file output structure has no stated reading order or summary-first entry point. A brief header in `strategic_briefing.md` ("Read this first. Then `scenario_library.md` for risk cases. `report.md` is the data appendix.") would resolve this.

5. **[P2] Approved vs. investigational distinction unclear in competitive density narrative.** The NF2-NSCLC ADC section did not distinguish approved ADCs (entrenched standard of care) from investigational ADCs (future competitive risk). This creates ambiguity that a sophisticated decider will flag. The output should be explicit about regulatory status when characterizing competitive density.
